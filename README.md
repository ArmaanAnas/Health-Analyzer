# Health Analyzer Project

A web-based health report analyzer built using:
- HTML, CSS, JavaScript
- Python Flask
- SQLite Database

## Features
- User registration & login
- Report analysis
- Detailed health summary
- BMI, Sugar, BP analysis
- Trend charts (Chart.js)
- Delete single / all reports
- CSV export
